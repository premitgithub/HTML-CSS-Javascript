Selectors in HTML

Element Selector

The element selector selects HTML elements based on the element name.

ID Selector

The ID Selector uses the id attributes of an HTML element to select a specified element. The ID of an element is unique within a page, soID selector is used to select one unique element.

Class Selector

The class selector selects HTML elements with a specific class attribute. To select elements with a specific class, write a period (.) character, followed by a calss name.
